Benvenuto nella documentazione utente di Epas¶

ePAS (Electronic Personnel Attendance System) è un software per la rilevazione e gestione delle presenze del personale sviluppato dall’Istituto di Informatica e Telematica (IIT-CNR) per essere utilizzato dagli istituti del Consiglio Nazionale delle Ricerche (CNR) Italiano. Questo documento descrive le principali funzionalità di ePAS dal punto di vista del dipendente.

Contenuti

  • Situazione mensile
  • Assenze mensili
  • Assenze annuali
  • Ferie
  • Competenze
  • Riepilogo orario
  • Ore di formazione
  • Gestione password
